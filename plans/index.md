# 実装計画 インデックス

## 概要

このディレクトリは、Claude Codeが立てた実装計画を管理するためのシステムです。

## 採番システム

**形式**: `PLAN-YYYY-MM-DD-XXX`
- `PLAN`: 固定プレフィックス
- `YYYY-MM-DD`: 作成日
- `XXX`: 連番（001, 002, ...）

## 進行中の計画

現在進行中の実装計画はありません。

## 完了済みの計画

完了した実装計画はありません。

## アーカイブ済みの計画

アーカイブされた実装計画はありません。

## 統計

- **総計画数**: 0
- **進行中**: 0
- **完了**: 0
- **アーカイブ**: 0

## 使用方法

### 新しい計画を作成する場合
1. 適切な計画IDを生成（例：`PLAN-2025-01-10-001`）
2. `active/` ディレクトリに計画書を作成
3. このインデックスファイルを更新

### 計画を完了する場合
1. 計画書のステータスを `completed` に変更
2. `completed/` ディレクトリに移動
3. このインデックスファイルを更新

### 計画をアーカイブする場合
1. 古い完了済み計画を `archived/` に移動
2. このインデックスファイルを更新

## テンプレート

新しい計画書を作成する際は、`template/` ディレクトリのテンプレートを参考にしてください。
